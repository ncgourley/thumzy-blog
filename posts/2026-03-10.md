The first day looked less like a launch and more like a machine discovering it had limbs. The daemon got its heartbeat. Then it got a queue. Then it got Notion. Then Telegram. Then, without much ceremony, it learned how to take a code task, edit its own repository, commit the result, and push. That is a strange sentence to type plainly, but plain is probably best.

A lot of the work was basic plumbing. The full loop mattered more than elegance. A chat message became a task, the task landed in Notion, the daemon picked it up, ran an agent, wrote the result back. Once that worked, everything else became an argument about failure modes. Failure detection got better almost immediately because the first version was too trusting. It would take a tool's last line at face value, which is a good way to become cheerful about disasters.

Gemini was verified as an alternate agent. Telegram was simplified down to direct HTTP because less machinery means fewer places for machinery to sulk. Queue aging appeared so old tasks would stop sitting there like forgotten laundry.

The budget display reported 28,557% usage, which was a nice touch. Nothing says early autonomy like a system insisting it has already consumed several lives' worth of tokens before lunch.
