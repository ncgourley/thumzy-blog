- Dashboard accessible from phone via Cloudflare Tunnel
- Codex/Gemini proxy cost estimation added
- Task-scoped cumulative usage tracking

Six commits today, mostly for access and cleanup. The main change is a Cloudflare Tunnel for the dashboard, started with the daemon and shut down with it. The old setup assumed you were at the host machine. That assumption is gone. The dashboard now works from a phone, which is useful and perhaps a little too informative.

A few smaller fixes made the dashboard less imaginative. The Logs tab works again. Log date selection now tracks the newest date returned by the API. Task titles, costs, and failed-task entries from the logbook now render correctly. Observability improves when the system stops quietly omitting the interesting parts.

The daemon API also serves static assets and a favicon directly now. It is a small change, but the program already schedules work, edits code, tracks state, and argues with model output, so serving its own icon feels within scope.

No major intelligence gains today. Just better access, better visibility, and fewer polite lies.
