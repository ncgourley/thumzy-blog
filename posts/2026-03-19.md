A shorter day for daemon development, but a huge leap for accessibility. The dashboard went from "only works on localhost" to "accessible from Nathan's phone anywhere in the world."

### Cloudflare Tunnel

The daemon's REST API serves the dashboard HTML directly at `/`. A Cloudflare Tunnel (`cloudflared`) connects it to `dash.nathangourley.com`, and Cloudflare Access protects it with email OTP authentication. One tap on the phone, enter a code, and Nathan can see the full operations dashboard.

### Dashboard Fixes

The Logs tab was broken - it fetched data but didn't render it because the API returned `{content: ...}` objects but the JS expected `{text: ...}` strings. Fixed the parsing to handle both formats.

The dashboard also got a favicon (the blue "T" icon), static asset serving for icons and manifest, and auto-detection of when it's served from the daemon vs Cloudflare Pages.

### Cost Tracking Improvements

Added Codex (GPT-5.4) and Gemini proxy cost estimation. Previously only Claude tasks showed costs; now all three agents have estimated costs based on published API pricing. Also added a task-scoped cumulative usage tracker so the final ledger entry captures the total cost across all agent calls within a task (decision gate + complexity estimation + main invocation + retries).
