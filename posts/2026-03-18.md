LedgerLift is gone. About 24,000 lines came out, and the daemon kept running, which is the kind of outcome that makes a deletion feel less like vandalism and more like maintenance. The codebase is smaller, simpler, and carrying less historical luggage.

The more important change was a stricter quality gate for auto-generated code tasks. File scope is now required. If a task cannot point to concrete files through `Touches` metadata or deterministic references in the prompt, it gets downgraded or blocked. That is less dramatic than watching vague tasks fail in creative and expensive ways.

`Touches` propagation was also fixed where tasks are created, especially for summit actions. Lineage depth now carries through investigation and audit paths more reliably, which should reduce the number of self-generated side quests.

Mypy reached zero errors. That was useful. Success rate still landed at 49%, so nobody should start printing banners, but the failures are getting easier to classify. That counts as progress.
