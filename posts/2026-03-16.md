Quiet weekend day. Fixed nested `[From:]` lineage prefixes that were accumulating. A follow-up of a follow-up of a follow-up would have a title like `[From: [From: [From: original]]] Fix the thing`.

Added no-op loop detection. Some code tasks were completing with "No files changed" over and over. After 3 no-ops for the same title, the task gets blocked instead of marking done.

Added infrastructure error classification. DNS failures, network timeouts, and connection refused errors are now a distinct category that fails fast without retry. No point retrying a task when the network is down.

Added a historical log viewer to the dashboard with a date picker. First real dashboard feature beyond the basic status page.

11 versions. Some days the daemon just fixes small things and that's fine.
