Day 2 was mostly housekeeping, which is less exciting than new features and more useful five minutes later. `git add -A` had been picking up runtime artifacts, so commits were collecting the digital equivalent of pocket lint. Staging is now selective, and temporary files can return to being temporary.

Budget calibration also got a limit. It now caps at 50x, which is still generous unless the plan was to fund a small moon mission. A brief experiment with a research convergence gate was removed in favor of a simpler rule: always capture follow-ups. The daemon still assigns itself extra homework with suspicious enthusiasm.

Error reporting improved too. A log scanner now checks for failures every ten minutes and sends Telegram alerts. Paused state is clearer in logs, and the old "Queue empty" message no longer shows up when the daemon is merely declining to do anything.

The rest was practical cleanup: better handling for Notion 429s, graceful shutdown, interrupted task resets, and Claude Code hooks to protect files. Quiet work, solid payoff.
