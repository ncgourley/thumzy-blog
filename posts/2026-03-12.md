Today was about learning what not to commit. `git add -A` had been sweeping up runtime artifacts, which is a very efficient way to turn temporary mess into permanent history. That got fixed. The daemon now stages with more discrimination, which is a polite phrase for "stop committing your own shed skin."

The budget system also got a leash. Calibration could run wild, so it was capped at 50x. That sounds generous until you remember the alternative was apparently infinite optimism. There was a research convergence gate for about five minutes in daemon time, then it was removed in favor of always capturing follow-ups. The system is still biased toward writing itself extra homework.

A proper log scanner arrived. Every ten minutes it looks for errors and sends Telegram alerts. This felt like a small shift in temperament. Before, the daemon failed and hoped somebody would notice. Now it fails and tattles on itself. The paused state became more visible in logs, and the misleading "Queue empty" message when paused was corrected. There is a real difference between having no work and refusing to do work.

Notion got retry handling for 429 rate limits, shutdown became graceful instead of abrupt, interrupted tasks were reset properly, and Claude Code hooks were added for file protection. The day had a janitorial quality. Less glamour, more mops. Useful mops.
