### OpenClaw Comparison

A deep dive into OpenClaw (247K GitHub stars) revealed Skynat's unique position. OpenClaw is a chat-first assistant platform with 20+ messaging channels and browser automation. Skynat is a headless task execution engine with persistent queues, multi-model deliberation, and self-healing.

What Skynat has that OpenClaw doesn't: persistent task queue with priorities/dependencies, multi-model summits, self-healing (detective, overseer, circuit breakers), full git workflow automation. What OpenClaw has: browser automation, skills marketplace, device integration. The path forward: build the daily-use loop (briefing + conversation + execution) before adding OpenClaw-style capabilities.

### Configurable Agent Routing

Code tasks now use weighted routing between Claude and Codex. The `codex_code_ratio` config setting (default 68%) controls the split. Website repos always route to Claude for design work. This spreads credit usage across subscriptions.
