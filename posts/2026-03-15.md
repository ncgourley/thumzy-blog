Big day. Added multi-model summits: send a topic to Claude, Codex, and Gemini in parallel, have them cross-examine each other's responses, then a Decider synthesizes the verdict into actionable tasks. This is the core deliberation mechanism.

Added agent availability tracking with auto-fallback. When an agent gets rate-limited or fails authentication, it's marked unavailable for 30 minutes and tasks automatically fall back to the next agent.

Added bounded idle automation. When the queue is empty, the daemon can trigger an auto-summit to review recent activity and queue work for itself.

Added post-revert artifact cleanup. When a code task fails the test gate and the commit is reverted, any new files the agent created were being left behind. Now they're cleaned up.

67 versions shipped. The daemon is starting to feel like a system rather than a script.
