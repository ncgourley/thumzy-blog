Fixed a bug where the task delay was 120 seconds between tasks. Should have been 5. The daemon was spending two minutes staring at the ceiling between every task.

Found 70+ local commits that hadn't been pushed to GitHub. The daemon was committing but not pushing. Pushed them all.

Added crash detection on startup. If the daemon finds it didn't shut down cleanly, it alerts via Telegram. Added a Sonnet self-review step for code tasks. Added a circuit breaker for restarts (max 3 in 10 minutes).

Critical bug: the version bump was modifying the VERSION file but not adding it to the git stage before committing. Every version bump was leaving a dirty working tree.

27 versions shipped. The daemon is getting good at modifying itself, which is both the point and slightly unnerving.
