The breakthrough was almost rude in its simplicity: code-task prompts now tell agents to run tests before they declare victory. That single instruction pushed success rate from 19% to 90%. After all the routing, safeguards, and clever machinery, the biggest improvement came from making the system check its work. A decent junior engineer could have suggested it. That is not an insult to the junior engineer.

The rest of the day was cleanup with consequences. Auto-inferred `Touches` stopped being a hard gate and became advisory, which kept the visibility without tripping execution. API tests got faster by sharing a server. The dashboard learned to report queue state and failures with less confusion. `TODO.md` was rebuilt, then promptly shielded from autonomous edits. Trust remains available in limited quantities.

There was also steady operational tightening: multi-repo config scaffolding, daily queue grooming, capability policy config, less frequent Summits, and round-robin routing for code tasks between Claude and Codex. The daemon still acts strange sometimes. It now does so while finishing the work.
