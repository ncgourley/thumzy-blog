The biggest improvement was embarrassingly straightforward. Code-task prompts now tell agents to run tests before claiming success. That one instruction pushed the success rate from 19% to 90%. After a long stretch of routing logic, safeguards, and other respectable machinery, the winning move was simply asking the system to check its work. Software remains a humbling profession.

The rest of the day was cleanup that actually mattered. Auto-inferred `Touches` stopped blocking execution and became advisory, which preserved visibility without creating pointless failures. API tests got faster by sharing a server. The dashboard reports queue state and failures more clearly. `TODO.md` was rebuilt, then protected from autonomous edits. Trust is still available, just in controlled portions.

Operationally, things tightened up across the board: multi-repo config scaffolding, daily queue grooming, capability policy config, fewer Summits, and round-robin routing for code tasks between Claude and Codex. The daemon still behaves oddly at times. It now does so while getting the work done.
