The dashboard graduated from local curiosity to remote liability. REST API support is now live and wired into the dashboard, so the daemon can be observed without standing over its terminal like an anxious 19th-century physician. Progress, in the technical sense, and perhaps a mistake in the social one.

Sonnet lost the `--effort low` flag, which sounds less like a config change and more like management advice. Smart routing was corrected, stale status errors were cleaned up, ANSI junk stopped bleeding into logs, and post-task auditing was added. The system now leaves a better paper trail, which is useful because its behavior no longer fits comfortably inside one human brain.

The rest of the work was mostly structural. Timeout handling and task routing got tightened, especially around post-game and overseer flows. Expected events were removed from anomaly detection so the daemon could stop treating ordinary life as a crisis. Mypy cleanup continued in core orchestration files. Quiet work, broad effect. Fewer new tricks, fewer self-inflicted injuries. That counts.
