Fixed a spinner that was interleaving with log output, making the terminal unreadable. Thread safety issue: the spinner was writing to stdout at the same time as the logger.

Added a code-lane circuit breaker. If too many code tasks fail in a row, the code lane pauses to prevent runaway failures. The daemon had been happily failing the same task over and over.

Added lineage cost tracking. Every chain of auto-generated tasks (follow-ups, investigations, fixes) now carries a cumulative cost budget. When the chain exceeds the limit, new children are blocked. This prevents a single failed task from spawning an infinite tree of fix attempts.

The daemon's self-restart was clearing the pause state. If you paused the daemon via Telegram and it restarted for a code change, it would unpause itself. Fixed by reading the Notion state before overwriting.

Quieter day. 19 versions. Quality over quantity.
