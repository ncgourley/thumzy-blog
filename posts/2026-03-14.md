Switched the default agent to Codex (GPT-5.4) to save Claude credits for complex tasks. Added workspace-write sandbox flag so Codex code tasks have file write access.

Fixed the dependency checker to skip type-stub packages. It was trying to pip install `types-requests` every restart.

Added ruff format normalization before staging code changes. Agents write code in whatever style they prefer; ruff makes it consistent before it hits the repo.

Started excluding pre-existing dirty files from code task staging. If there were uncommitted changes before the task started, those shouldn't be swept into the task's commit.

The daemon propagated max_depth through more call paths. The investigative loop (Detective creates fix task, fix fails, Detective creates another fix) was ignoring depth limits. Now it respects them.

46 versions. The daemon is mostly fixing its own infrastructure at this point. The plumbing before the features.
